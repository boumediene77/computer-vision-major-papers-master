# Computer Vision Major Papers
This repository contains an implementation of the contributions presented in the following research papers using PyTorch:

- [Intriguing properties of neural networks](https://arxiv.org/abs/1312.6199)
- [Visualising image classification models and saliency maps](https://arxiv.org/abs/1312.6034)
- [Understanding Neural Networks Through Deep Visualization](https://arxiv.org/abs/1506.06579)
- [Unsupervised Representation Learning With Deep Convolutional Generative Adversarial Networks](https://arxiv.org/abs/1511.06434)
- [Conditional Generative Adversarial Nets](https://arxiv.org/abs/1411.1784)
